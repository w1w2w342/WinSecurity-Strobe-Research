Описание программы

Это программа для Windows написанная на Python с использованием tkinter PIL OpenCV и keyboard
Она делает окно на весь экран которое невозможно закрыть обычными способами
При запуске программа добавляется в автозагрузку Windows
Окно блокирует клавиши Windows Alt Tab Ctrl Shift и другие чтобы нельзя было переключаться
В окне воспроизводится видео с текстовыми подсказками и звуковой сигнал
Пользователь может разблокировать окно введя правильный код в поле ввода и нажав кнопку

После разблокировки окно исчезает музыка останавливается клавиатура разблокируется
Программа начинает бесконечно менять обои рабочего стола с использованием заранее подготовленных картинок
Смена обоев происходит каждые доли секунды и работает в фоновом потоке
Функции

resource_path
Возвращает путь к файлу независимо от того запускается программа как EXE или как скрипт

add_to_registry
Добавляет программу в автозагрузку

remove_from_registry
Удаляет программу из автозагрузки

block_event
Блокирует стандартное закрытие окна и Alt F4

stream_video
Потоковое воспроизведение видео с наложением текста

unlock
Проверяет введенный код и разблокирует программу останавливает музыку снимает блокировку клавиш запускает смену обоев

start_crazy_wallpapers
Бесконечно меняет обои используя список изображений

run_in_background
Запускает смену обоев в отдельном потоке

Как использовать

1 Скопировать все файлы видео изображения и аудио в одну папку с программой
2 Запустить скрипт на Windows
3 Программа откроет полноэкранное окно и заблокирует клавиатуру
4 Для разблокировки ввести код 67 и нажать кнопку Unlock
5 После разблокировки программа начнет менять обои в фоне
